# V5.5 İçe / Dışa Aktar Kullanımı

## Dışa Aktar
Merkez ekranında:
1. Tarih seç
2. Sol menüden `İçe / Dışa Aktar`
3. `Seçili Tarihi Dışa Aktar` butonuna bas
4. Excel dosyası iner

## İçe Aktar
1. `Örnek Excel Şablonu İndir` butonuna bas
2. Excel dosyasını doldur
3. Kolon adlarını değiştirme
4. `Dosya Seç` ile Excel'i yükle
5. Önizlemeyi kontrol et
6. `Önizlenen Kayıtları İçe Aktar` butonuna bas

## Zorunlu Kolonlar

TIME
HOTEL
PAX_ADULT
PAX_CHILD
NATION

## Opsiyonel Kolonlar

## Saat Formatı
